# Aplikasi Pengolahan Citra Digital

Aplikasi desktop berbasis Python untuk pengolahan citra digital menggunakan Tkinter sebagai antarmuka dan OpenCV sebagai pemrosesan gambar.

## Fitur
- Upload gambar (JPG, PNG, JPEG)
- Grayscale
- Smoothing
- Sharpening
- Emboss
- Informasi gambar (nama file, ukuran, mode warna, operasi)

## Library
- Python 3
- Tkinter
- OpenCV
- Pillow (PIL)
- NumPy

## Cara Instalasi
- Clone repository ke laptop
- Install library `opencv-python`, `pillow`, dan `numpy`
- Jalankan aplikasi dengan perintah `python main.py`

## Cara Penggunaan
- Jalankan aplikasi
- Unggah gambar yang akan diproses
- Pilih operasi pengolahan citra yang tersedia
- Hasil pengolahan dan informasi gambar akan ditampilkan pada aplikasi
